# 반응형 웹 사이트

  • 화면 크기에 따라 다르게 보이는 웹 사이트
  • 사용자에게 화면 크기에 따라 최적화된 스타일을 제공할 수 있다
  • @media 문법을 사용한다
  • @media 미디어타입 and 원하는 크기

# 사용 가능한 미디어 종류

  • all : 모든 종류의 미디어
  • print : 프린터로 출력 할때의 스타일 절정
  • screen : PC, 태블릿PC, 스마틒ㄴ 등의 화면을 보는 일반 기기
  • speech : 화면을 읽어주는 기기
안녕하세요
