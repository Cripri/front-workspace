# Browser Object Model (BOM)

  • window 객체
  • 웹 브라우저에 직업 접근하여 웹 브라우저의 기능을 활용할 수 있는 객체
  • DOM도 window 내부에 존재하는 객체이다 (window.document)
  • window는 항상 생략이 가능하다

# BOM 내부의 여러 객체들

  • window : 웹 브라우저 최상위 객체
  • window.documeny : 현재 페이지의 HTML구조를 담고있는 객체
  • window.history : 웹 브라우저의 페이지 이동 내역을 담고있는 객체
  • window.screen : 웹 브라우저가 표시되는 화면에 대한 정보를 담고있는 객체
  • window.location : 웹 브라우저가 현재 보고있는 위치(경로)에 대한 정보
  • window.navigator : 웹 브라우저 자체에 대한 정보
