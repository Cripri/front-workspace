# 자바스크립트에서 함수 사용하가

  • function 키워드를 사용해 함수를 선언할 수 있다
  • 선언한 함수를 ()를 붙이지 않고 활용할 수도 있다
  • 함수 이름 뒤에 ()를 붙여 호출하면 해당 함수의 실행 결과를 리턴하고 ()를 붙이지 않고 사용하면 함수 자체를 전달하는 것이된다
  • 리턴타입을 적지 않아도 값을 리턴할 수 있다
  • 자바스크립트는 변수에 함수를 저장할 수 있다

# 자바스크립트 함수의 종료

  • function으로 선언해 사용하는 함수
  • 함수 표현식: 변수에 함수를 담아서 사용하는 함수
  • 화살표 함수 : 함수 표현식을 좀더 멋있고 감편하게 사용하기 위한 함수
